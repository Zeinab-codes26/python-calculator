# Python Calculator 

A simple calculator application built with Python and Object-Oriented Programming.

## Features
- Addition
- Subtraction
- Multiplication
- Division
- Memory system
- Graphical User Interface with Tkinter

## Technologies
- Python
- Tkinter
- OOP
